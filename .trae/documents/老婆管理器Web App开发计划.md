# 技术栈选择
- **前端框架**: React + TypeScript (类型安全，组件化开发)
- **构建工具**: Vite (快速开发和构建)
- **样式方案**: Tailwind CSS (现代UI设计，响应式布局)
- **状态管理**: React Context API + useReducer (轻量级状态管理)
- **数据存储**: 本地文件系统 (Node.js fs模块) + 加密库 (确保数据安全)
- **认证方案**: 密码哈希存储 + 会话管理
- **图片处理**: FileReader API (本地图片上传和处理)

# 项目结构
1. **初始化项目**
   - 使用Vite创建React + TypeScript项目
   - 配置Tailwind CSS
   - 设置项目基础结构

2. **核心功能模块**
   - **认证系统**: 首次打开设置用户信息，后续使用密码登录
   - **用户档案**: 用户名、期望称呼、生日、密码、头像
   - **老婆档案**: 照片、姓名、生日、身高、体重、三维、纪念日，自动计算年龄和BMI
   - **图片画廊**: 轮播展示，批量上传，区分SFW/NSFW
   - **生理周期跟踪**: 安全期/经期计算，怀孕预警，可操状态显示，做爱记录
   - **消费记录**: 消费追踪，愿望单管理

3. **技术实现要点**
   - **数据加密**: 使用加密库对本地存储的数据进行加密
   - **跨浏览器访问**: 使用本地文件系统存储，确保数据不依赖于特定浏览器
   - **模块化设计**: 文本和样式支持模块化替换
   - **响应式布局**: 优先桌面使用，兼容移动端浏览
   - **UI风格**: 参考成人网站或Escort Agency的现代设计风格

4. **开发步骤**
   - 初始化项目和配置环境
   - 实现本地文件存储和加密功能
   - 开发认证系统（首次设置和密码登录）
   - 实现核心布局和导航
   - 开发用户档案管理
   - 实现老婆档案管理
   - 构建图片画廊功能
   - 开发生理周期跟踪系统
   - 实现消费记录功能
   - 测试和优化
